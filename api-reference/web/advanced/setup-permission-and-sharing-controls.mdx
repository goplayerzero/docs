### Roles in PlayerZero

1. Member: Can view data settings & add team members. Anybody who joins your
   project will be a member by default and can be promoted to Owner if needed in
   the sharing page.
2. Owner: Can view, add, edit, and delete all settings.

### Setup Steps

1. Sign-up for PlayerZero
2. Create your first project
3. Go to Settings > Team
4. Navigate to the "My Team" section and click "Invite"
5. Add your teammates' emails
