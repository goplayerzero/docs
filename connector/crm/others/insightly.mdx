[Insightly](https://www.insightly.com/) is a CRM and project management software for businesses of all sizes, helping them build lasting relationships.

## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Click on the “Connectors” button.
3. Filter by the ”CRM” group in the dropdown.
4. Search for “Insightly”.
5. Login to your Insightly account if not already logged in.
6. Enter your Insightly project domain or the appropriate API key for your project.
7. Authorize Merge to read and write data through the Insightly connector.
