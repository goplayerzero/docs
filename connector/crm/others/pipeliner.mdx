[Pipeliner](https://www.pipelinersales.com/) is a sales CRM software with an emphasis on pipeline management, sales processes, and analytics.

## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Click on the “Connectors” button.
3. Filter by the ”CRM” group in the dropdown.
4. Search for “Pipeliner”.
5. Login to your Pipeliner account if not already logged in.
6. Enter your Pipeliner project domain or the appropriate API key for your project.
7. Authorize Merge to read and write data through the Pipeliner connector.
