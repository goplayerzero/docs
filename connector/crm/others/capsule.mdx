[Capsule](https://capsulecrm.com/) is a simple, effective, and flexible CRM software for managing customer relationships and sales pipelines.

## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Click on the “Connectors” button.
3. Filter by the ”CRM” group in the dropdown.
4. Search for “Capsule”.
5. Login to your Capsule account if not already logged in.
6. Enter your Capsule project domain or the appropriate API key for your project.
7. Authorize Merge to read and write data through the Capsule connector.
