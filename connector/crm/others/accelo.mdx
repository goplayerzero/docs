[Accelo](https://www.accelo.com/) is a service operations automation tool to run your entire service business, from prospect to payment.

## Setup

1. From the PlayerZero web app, navigate to the home page.
2. Click on the “Connectors” button.
3. Filter by the ”CRM” group in the dropdown.
4. Search for “Accelo”.
5. Login to your Accelo account if not already logged in.
6. Enter your Accelo project domain or the appropriate API key for your project.
7. Authorize Merge to read and write data through the Accelo connector.
