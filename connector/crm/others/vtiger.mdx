[Vtiger](https://www.vtiger.com/) is a CRM software that helps more than 300,000 businesses grow sales, improve marketing reach, and deliver great customer service.

## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Click on the “Connectors” button.
3. Filter by the ”CRM” group in the dropdown.
4. Search for “Vtiger”.
5. Login to your Vtiger account if not already logged in.
6. Enter your Vtiger project domain or the appropriate API key for your project.
7. Authorize Merge to read and write data through the Vtiger connector.
