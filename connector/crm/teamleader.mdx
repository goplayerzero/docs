[Teamleader](https://www.teamleader.eu/) is a unified CRM, invoicing, and project planning tool for SMEs.

## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Click on the “Connectors” button.
3. Filter by the ”CRM” group in the dropdown.
4. Search for “Teamleader”.
5. Login to your Teamleader account if not already logged in.
6. Enter your Teamleader project domain or the appropriate API key for your project.
7. Authorize Merge to read and write data through the Teamleader connector.
