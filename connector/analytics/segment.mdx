[Segment](https://segment.com/) is a customer data platform that collects, stores, and routes user data to various analytics and marketing tools. PlayerZero implements with both Segment Web and Cloud solutions.

<Snippet file="telemetry-connector.mdx" />

## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Click on the “Connectors” button 
3. Filter by the ”Product analytics” group in the dropdown
4. Search for and select “Segment ”
5. Create a PlayerZero [destination in Segment](https://app.segment.com/goto-my-workspace/destinations/catalog/actions-playerzero-cloud)
6. Copy & paste your project id / token 
7. Enable the destination and save your changes.
