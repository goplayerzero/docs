## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Select the Re:amaze connector under the Tickets section.
3. Login to your Re:amaze account if not already logged in.
4. Enter your Re:amaze project domain or the appropriate API key for your project.
5. Authorize Merge to read and write data through the Re:amaze connector.
