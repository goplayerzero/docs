[Freshdesk](https://www.freshworks.com/freshdesk/) is an online cloud-based customer service software providing helpdesk support with all smart automations.

<Snippet file="ticketing-connector.mdx" />

## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Select the Freshdesk connector under the Tickets section.
3. Login to your Freshdesk account if not already logged in.
4. Enter your Freshdesk project domain or the appropriate API key for your project.
5. Authorize Merge to read and write data through the Freshdesk connector.
