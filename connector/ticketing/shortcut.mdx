## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Select the Shortcut connector under the Tickets section.
3. Login to your Shortcut account if not already logged in.
4. Enter your Shortcut project domain or the appropriate API key for your project.
5. Authorize Merge to read and write data through the Shortcut connector.
