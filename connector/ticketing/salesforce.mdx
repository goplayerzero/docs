## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Select the Salesforce Service Cloud connector under the Tickets section.
3. Login to your Salesforce Service Cloud account if not already logged in.
4. Enter your Salesforce Service Cloud project domain or the appropriate API key for your project.
5. Authorize Merge to read and write data through the Salesforce Service Cloud connector.
