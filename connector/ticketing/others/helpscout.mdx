[Helpscout](https://www.helpscout.com/) is a customer service software company providing an email-based customer support platform, knowledge base tool, and an embeddable search/contact widget.

<Snippet file="ticketing-connector.mdx" />

## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Select the Helpscout connector under the Tickets section.
3. Login to your Helpscout account if not already logged in.
4. Enter your Helpscout project domain or the appropriate API key for your project.
5. Authorize Merge to read and write data through the Helpscout connector.
