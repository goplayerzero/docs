[Zoho BugTracker](https://www.zoho.com/bugtracker/) is a bug tracking software that helps you manage software defects and resolve issues quickly.

<Snippet file="ticketing-connector.mdx" />

## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Select the Zoho BugTracker connector under the Tickets section.
3. Login to your Zoho BugTracker account if not already logged in.
4. Enter your Zoho BugTracker project domain or the appropriate API key for your project.
5. Authorize Merge to read and write data through the Zoho BugTracker connector.
