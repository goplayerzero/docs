## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Click on the “Connectors” button.
3. Filter by the ”Project Management” group in the dropdown.
4. Search for “Teamwork”.
5. Login to your Teamwork account if not already logged in.
6. Enter your Teamwork project domain or the appropriate API key for your project.
7. Authorize Merge to read and write data through the Teamwork connector.
