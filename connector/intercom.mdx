Intercom is a customer messaging platform that allows businesses to communicate with prospective and existing customers within their app, on their website, through social media, or via email.

## What You Can Do

Once connected, PlayerZero AI agents can:

- **Get tickets** — Access full ticket details including title, description, state, and attributes
- **Search tickets** — Find tickets using filters across fields like state, assignee, category, and dates with pagination support
- **Reply to tickets** — Post admin comments directly on tickets with HTML formatting support
- **Get conversations** — Read full conversation details and message threads
- **Search conversations** — Find conversations using multi-filter search with pagination
- **Reply to conversations** — Post replies visible to customers, internal admin notes, or assignment updates on conversations

## Use Cases

- **Automated triage** — AI agents analyze incoming Intercom conversations and surface relevant code context, related tickets, or known issues
- **Debug-to-ticket workflow** — Convert debug findings into well-documented Intercom replies with root cause analysis
- **Cross-reference analysis** — When investigating bugs, agents pull in related Intercom conversations for full customer impact context

## Setup

<Note>
Setup starts in PlayerZero — no pre-configuration in Intercom is required.
</Note>

1. In PlayerZero, navigate to **Organization Settings → Integrations**
2. Find **Intercom** and click **Connect**
3. You'll be redirected to Intercom to log in and authorize PlayerZero
4. After authorization, select which PlayerZero projects should have access to this integration
