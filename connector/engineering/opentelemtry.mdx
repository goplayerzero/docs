[OpenTelemetry](https://opentelemetry.io/) (also referred to as OTel) is an open source observability framework made up of a collection of tools, APIs, and SDKs. OTel enables IT teams to instrument, generate, collect, and export telemetry data for analysis and to understand software performance and behavior.

## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Click on the “Connectors” button 
3. Filter by the ”Engineering” group in the dropdown
4. Search for and select “Opentelemetry”
5. Add the project specific endpoint to your config.yml file
  1. If not using a config file, visit the Opentelemetry [docs](https://opentelemetry.io/docs/concepts/sdk-configuration/otlp-exporter-configuration/) page endpoint configuration tips
