[Sentry](https://sentry.io/welcome/) is a cross-platform application monitoring tool with a focus on error tracking.

<Snippet file="engineering-connector.mdx" />

## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Click on the “Connectors” button 
3. Filter by the ”Engineering” group in the dropdown
4. Search for and select “Sentry”
5. Input your [Sentry Organization Slug](https://sentry.io/settings/organization/)
6. Input your [Sentry Auth Token](https://sentry.io/api/)
