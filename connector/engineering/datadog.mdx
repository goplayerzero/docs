[Datadog](https://www.datadoghq.com/) is a monitoring service for cloud-scale applications, providing monitoring of servers, databases, tools, and services.

<Snippet file="engineering-connector.mdx" />

## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Click on the “Connectors” button 
3. Filter by the ”Engineering” group in the dropdown
4. Search for and select “Datadog”
5. Input your [Datadog API Key](https://docs.datadoghq.com/account_management/api-app-keys/#api-keys)
6. Input your [Datadog Application Key](https://docs.datadoghq.com/account_management/api-app-keys/#application-keys)
