[AWS Cloudwatch](https://aws.amazon.com/cloudwatch/) is a monitoring and management service provided by Amazon for the AWS cloud and the applications running on AWS.

<Snippet file="engineering-connector.mdx" />

## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Click on the “Connectors” button 
3. Filter by the ”Engineering” group in the dropdown
4. Search for and select “AWS Cloudwatch”
5. Input your [AWS Access Key ID](https://docs.aws.amazon.com/IAM/latest/UserGuide/id_users_create.html)
6. Input your [AWS Secret](https://docs.aws.amazon.com/IAM/latest/UserGuide/id_credentials_access-keys.html)
7. Add your AWS Region
