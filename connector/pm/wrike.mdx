[Wrike](https://www.wrike.com/) is a cloud-based project management software for managing and tracking projects, deadlines, schedules, and other workflow processes.

<Snippet file="pm-connector.mdx" />

## Setup
1. From the PlayerZero web app, navigate to the home page.
2. Click on the “Connectors” button.
3. Filter by the ”Project Management” group in the dropdown.
4. Search for “Wrike”.
5. Login to your Wrike account if not already logged in.
6. Enter your Wrike project domain or the appropriate API key for your project.
7. Authorize Merge to read and write data through the Wrike connector.
