## Setup
1. From the PlayerZero web app, navigate to the home page and click `Connectors`.
2. Select the Jira Software connector under the Tickets section.
3. Login to your Jira Software account if not already logged in.
4. Enter your Jira Software project domain or the appropriate API key for your project.
5. Authorize Merge to read and write data through the Jira Software connector.
