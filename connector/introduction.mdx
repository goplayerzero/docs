PlayerZero integrates with your existing stack through OAuth-based connections. Once connected, AI agents can read data from your tools, take actions on your behalf, and provide context-aware assistance.

## What Integrations Enable

- **Read context** - AI agents access tickets, issues, documentation, and customer data when investigating problems
- **Take actions** - Create issues, post comments, update statuses, and escalate findings to the right teams
- **Cross-reference** - Search related records across tools to identify patterns and known issues

## Supported Integrations

| Integration | Category | What It Enables |
|-------------|----------|-----------------|
| [Azure DevOps](/connector/azure-devops) | Project Management | Work items, boards, iterations |
| [Confluence](/connector/confluence) | Documentation | Docs, runbooks, architecture |
| [HubSpot](/connector/hubspot) | CRM | Contacts, companies, tickets |
| [Intercom](/connector/intercom) | Support | Conversations, tickets, triage |
| [Jira](/connector/jira) | Project Management | Issue tracking, sprint context |
| [Linear](/connector/linear) | Project Management | Issue tracking, triage, comments |
| [Salesforce](/connector/salesforce) | CRM | Accounts, cases, contacts |
| [ServiceNow](/connector/servicenow) | IT Service | Incidents, changes, CMDB |
| [Zendesk](/connector/zendesk) | Support | Tickets, customer history |

<Note>
**Slack** is also available as a connector but is set up separately via **Organization Settings → Notifications** (not the Integrations page). See [Slack](/connector/slack) for full details on capabilities, shortcuts, and setup.
</Note>

## Setup

All integrations follow the same process:

1. Navigate to **Settings → Integrations**
2. Find the integration and click **Connect**
3. Authorize PlayerZero via OAuth
4. Select which projects to enable
